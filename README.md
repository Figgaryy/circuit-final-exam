# EES216 — AC Power & Three-Phase Circuits Study Guide

Interactive single-page study guide covering Chapter 11 (AC Power Analysis) and Chapter 12 (Three-Phase Circuits) for the EES216 final exam.

## Features

- **Chapter 11**: Instantaneous & average power, RMS, Complex Power, **Conservation of AC Power** (with circuit diagrams), PF Correction, Maximum Power Transfer
- **Chapter 12**: Y-Y, Y-Δ, Δ-Y, Δ-Δ connections + line-impedance examples
- **Interactive Wye↔Delta converter** — toggle and see what changes in real-time
- **3-Phase System Calculator** — pick configuration, get step-by-step solution
- **Y vs Δ Live Comparison** — same Z connected as Y vs Δ, see V<sub>phase</sub>, I<sub>line</sub>, P scaling
- **3 full-length exam-style problems** (Δ source + Y∥Δ parallel) with detailed a–f solutions
- Formula cheat sheet, MCQ practice, and worked examples throughout

## Usage

Open `remixed-91f23176.html` in any modern browser. No build step or server required.
